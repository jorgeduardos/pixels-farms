ABOUT (scroll)

Pixels-farms is a simple tool that helps farmers keep track of the crops they are currently farming

HOW TO USE

  • Add a farm using the fields above.
    • Type the farm number or just copy/paste the URL.
    • Select the crop type
    • Click "Add Farm"
    • A farm will appear on the left column
  • Start Farm
    • Click the "Start" button to start the timer on the farm
    • This will open the game in a new tab so you can go ahead and collec/plant the crop
    • Close the game and continue with the next farm in the list
    • Rows will turn green whenever the timer is over, press start to start the farm again
  • Edit Farm
    • Click the "Edit" button to edit the farm
    • A pop-up will open where you can select the new crop type for that farm
    • Click "Confirm" to confirm selection
  • Open Farm
    • Click the "Open" button to open Pixels game for that farm (this action wont start the timer)
  • Delete Farm
    • Click the "Delete" button to delete farm from the list
  • Bulk Editing
    • Bulk editing is possible by clicking "Select All" or by having two or more farms selected.
    • To select a farm click on the checkbox located on the left side of the farm number
  • Re-order Farms
    • Drag and drop farms to re-order them
  • Exporting/Importing Farms
    • To export farms click the "Export Farms" button
    • This will automatically download a txt file named "exported-farms"
    • To import farms click the "Import Farms" button
    • This will open a pop-up where you can copy and paste the contents located in an "exported-farms" txt file
  • Farms data is persisted even after refreshig the website :D
